Packageqnx.ui.media
Classpublic class MediaControl
InheritanceMediaControl UIComponent flash.display.MovieClip

The MediaControl class provides a set of controls that can be used to control media, such as video and audio.

The control itself does not control the media. Instead, it broadcasts events as the user interacts with the controls. You can customize the control by setting options to hide or show interace features.

setOption()method 
public function setOption(option:String, value:Boolean):void

Allows you to hide and show features of the MediaControl class.

By default all items are hidden.

Valid values for the option parameter are:

  • MediaControlOptions.VOLUME
  • MediaControlOptions.FULLSCREEN
  • MediaControlOptions.NEXT
  • MediaControlOptions.PREVIOUS
  • MediaControlOptions.PLAY_PAUSE
  • MediaControlOptions.SEEKBAR
  • MediaControlOptions.STOP
  • MediaControlOptions.DURATION
  • MediaControlOptions.POSITION

The MediaControlOptions.SCRUBBER option will be disabled if MediaControlProperty.DURATION property has not been set using the setProperty method.

Parameters

option:String — The option to show or hide.
 
value:Boolean — When set to true the option is shown. When set to false the option is hidden.

Dispatched when the state of the media should change based on user interaction with the controls.

The following cases will cause this event to be dispatched:

  • The user presses the stop button.
  • The user presses the pause button.
  • The user presses the play button.
  • The user touches down on the seek thumb.
  • The user releases the seek thumb.

Examples
The following example sets up a MediaControl object. 
        var controls:MediaControl = new MediaControl();
        controls.width = 995;
        controls.height = 65;
        addChild( controls );
        
        controls.setOption( MediaControlOptions.VOLUME, true );
        controls.setOption( MediaControlOptions.PLAY_PAUSE, true );
        controls.setOption( MediaControlOptions.NEXT, true );
        controls.setOption( MediaControlOptions.PREVIOUS, true );
        controls.setOption( MediaControlOptions.STOP, true );
        controls.setOption( MediaControlOptions.SEEKBAR, true );
        controls.setOption( MediaControlOptions.DURATION, true );
        controls.setOption( MediaControlOptions.POSITION, true );
        controls.setOption( MediaControlOptions.FULLSCREEN, true );
        controls.setProperty( MediaControlProperty.VOLUME, 80 );
        
        controls.addEventListener( MediaControlEvent.STATE_CHANGE, stateChanged );
        
        
        function stateChanged( e:MediaControlEvent ):void
        {
            var state:String = controls.getState();
            trace( state );
        }
