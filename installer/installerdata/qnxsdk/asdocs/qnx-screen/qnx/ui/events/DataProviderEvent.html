Packageqnx.ui.events
Classpublic class DataProviderEvent
InheritanceDataProviderEvent flash.events.Event

A DataProviderEvent object is dispatched into the event flow whenever a DataProvider change occurs. A change occurs through modification of the data provider by adding, removing, replacing, updating.
