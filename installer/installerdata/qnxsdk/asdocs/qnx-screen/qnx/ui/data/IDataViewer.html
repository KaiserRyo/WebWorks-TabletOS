Interfacepublic interface IDataViewer extends flash.events.IEventDispatcher
Implementors DropDown, List, Picker, RoundList, SegmentedControl

Interface to be implemented by components that provide access to dataProviders. The IDataViewer interface provides the functionality for manipulating data in lists.
