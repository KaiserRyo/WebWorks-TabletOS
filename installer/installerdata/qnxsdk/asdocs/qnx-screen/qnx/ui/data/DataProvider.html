Classpublic class DataProvider
InheritanceDataProvider flash.events.EventDispatcher
Implements IDataProvider
Subclasses SectionDataProvider

The DataProvider class provides a linear representation of a set of data and methods to enable modification of the data. List classes, such as the List and TileList use the DataProvider class in order to handle their data.
