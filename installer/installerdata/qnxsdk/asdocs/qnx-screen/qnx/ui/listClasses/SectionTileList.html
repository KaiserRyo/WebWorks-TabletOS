Classpublic class SectionTileList
InheritanceSectionTileList SectionList List UIComponent flash.display.MovieClip

The SectionTileList class allows you to group sections of data in a grid.

When the list is set to scroll vertically, the columnCount property is used to layout the grid. You can use the rowHeight and columnWidth height properties to set the dimensions of each cell. You can use the cellPadding property to set the padding between each cell.
