InheritanceListSelectionMode Object

The ListSelectionMode class defines modes that govern list selection. Each mode is defined by one of the following constants:
  • NONE - indicates no selection
  • SINGLE - indicates that a single item can be selected
  • MULTIPLE - indicates that multiple items can be selected
