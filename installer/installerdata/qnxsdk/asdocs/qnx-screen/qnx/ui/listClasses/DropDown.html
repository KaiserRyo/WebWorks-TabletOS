Packageqnx.ui.listClasses
Classpublic class DropDown
InheritanceDropDown UIComponent flash.display.MovieClip
Implements IDataViewer

The DropDown class provides drop down list functionality. When the user clicks on the down arrow, the list expands to provide options that a user can select from.

The following example shows a DropDown being instantiantated and initialized: 

     var dp:DropDown= new DropDown();
     dp.rowHeight = 45;
     dp.rowCount=3;
     addChild(dp);
     dp.width = 250;
         
     var dpp:Array=[];
     for(var i:int = 0;i<10;i++)
     {
       dpp.push({label:"test label"+i.toString()});
     }
     dp.dataProvider = new DataProvider(dpp);

An array of labels is created and set as the DataProvider for the DropDown.

promptproperty 
prompt:String

Gets or sets the prompt for the combox box.

The prompt is shown as the default message in the combo box when the selectedIndex value is set to -1. For example, the String "Select an item" prompts the user to select an item from the drop down list. If no prompt is set, the selectedIndex value is set to 0 and the first item in the DataProvider is displayed in the drop down.
