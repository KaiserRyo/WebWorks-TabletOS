setSkin()method
public function setSkin(cellSkin:Object):void

Set the skin object on the component for it to render.

If a skin is found, it is removed and a new skin is created. The cellSkin parameter can be one of three different types: Class, String, or DisplayObject instance.

  • If you pass in a Class, a new instance of that class is created and returned.
  • If you pass in a String, the flash.utils.getDefinitionByName will attempt to create a new method.
  • If you pass in a DisplayObject instance, the x and y values of the instance are set to 0 and returned.

Skins must implement the ISkin interface.
