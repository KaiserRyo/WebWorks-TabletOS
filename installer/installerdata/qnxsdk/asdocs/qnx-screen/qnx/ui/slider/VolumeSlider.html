Classpublic class VolumeSlider
InheritanceVolumeSlider Slider UIComponent flash.display.MovieClip

The volume slider component provides the functionality for implementing a volume control.

The volume slider differs from the base Slider class in that it has a specific look and feel and is used for media-based components.
