The RadioButton component lets you force a user to make a single selection from a set of choices. This component must be used in a group of at least two RadioButton instances. You can only select one member of the group at any given time. Selecting one radio button in a group deselects the currently selected radio button in the group. You can set the groupname parameter to indicate which group a radio button belongs to.

In the following code listing, three radio buttons are created and arranged on the background: 

     var rb1:RadioButton = new RadioButton();
     rb1.label = "Meal option 1";
     addChild( rb1 );
     
     var rb2:RadioButton = new RadioButton();
     rb2.label = "Meal option 2";
     addChild( rb2 );
     rb2.x = rb1.width + 10 + rb1.x;
         
     var rb3:RadioButton = new RadioButton();
     rb3.label = "Meal option 3";
     addChild( rb3 );
     rb3.x = rb2.width + 10 + rb2.x;

The x properties of each button are set using the height and width, and coordinate values of each previously created, adjacent button.
