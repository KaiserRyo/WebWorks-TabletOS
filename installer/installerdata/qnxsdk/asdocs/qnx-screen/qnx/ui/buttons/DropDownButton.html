Classpublic class DropDownButton
InheritanceDropDownButton LabelButton Button SkinnableComponent UIComponent flash.display.MovieClip

The DropDownButton class represents the top button in the DropDown list. It contains the label used to highlight the selected item. You can click on the DropDownButton to open and close the list.
