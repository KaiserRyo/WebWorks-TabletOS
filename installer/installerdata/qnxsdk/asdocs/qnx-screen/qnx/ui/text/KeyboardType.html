KeyboardTypeConstants
 
Packageqnx.ui.text
Classpublic class KeyboardType
InheritanceKeyboardType Object

The KeyboardType class contains constants that define the layout of the keyboard that appears when the user clicks in a textInput component. For example, you can enable an email-specific keyboard when the user clicks in a textInput component that takes an email address.

The following code listing creates an email-specific keyboard: 

     firstInput = new TextInput;
     firstInput.keyboardType = KeyboardType.EMAIL;
     firstInput.width = 200;
