Packageqnx.dialog
Classpublic class DialogButtonProperty
InheritanceDialogButtonProperty Object

The DialogButtonProperty class contains dialog button properties.

The properties enable you to modify the appearance or behavior of a button. In the following listing, a dialog button is disabled by passing the DialogButtonProperty.ENABLED property value (and the value false) into the setButtonPropertyAt method: 

        private function showAlertDialog():void
        {
           alert = new AlertDialog();
          alert.title = "Meeting reminder";
          alert.message = "The group benefits meeting in the Blue room is about to start.";
          alert.addButton("Snooze");
          alert.addButton("Cancel");
          alert.addButton("Help");
          alert.setButtonPropertyAt(DialogButtonProperty.ENABLED, false, 2);
          alert.dialogSize= DialogSize.SIZE_SMALL;
          alert.addEventListener(DialogEvent.DIALOG_BUTTON_CLICKED, dButtonClicked); 
          alert.show(IowWindow.getAirWindow().group);
         }
