Classpublic class DialogEvent
InheritanceDialogEvent flash.events.Event

The DialogEvent class provides user-related dialog event functionality. For example, the DialogEvent class contains the DIALOG_BUTTON_CLICKED event type that indicates when a user has clicked a button on a dialog.
