Skin assets

This topic provides a description of the Flash assets that are included with this SDK. All assets are defined in the qnx.ui.skins.SkinAssets class.

The following components are described in detail:

  • Lists
  • Text Input
  • Rounded Button
  • Check Box
  • Toggle
  • Drop Down
  • Drop Down Cell Renderer
  • Volume Controls
  • Slider
  • Picker

Volume controls

A volume control is comprised of two skins: track and thumb. The thumb is unique as it implements the IProgress interface. The fill for the volume control is a gradient that is drawn dynamically and can be accessed as a property on the volume control.

Volume track fill

To set the the volume track fill gradient, use the fillColors and disabledColors methods in the VolumeSlider class. The disabledColors method sets the gradient fill for the disabled state. Both methods take an array of two color values.

The default gradients are listed in the following tables:

Light color theme

State Value 1 Value 2
Up 0x05275b 0x0172c4
Disabled 0x2d2d2d 0x5f5f5f

Dark color theme

State Value 1 Value 2
Up 0x2d2d2d 0x5f5f5f
Disabled 0x2d2d2d 0x5f5f5f
